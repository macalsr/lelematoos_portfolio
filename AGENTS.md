# AGENTS.md

## Objetivo
Implementar e evoluir o site da Lele Matoos em Next.js + TypeScript + Tailwind.

## Regras de design
- O HTML/layout aprovado pelo usuário é a fonte de verdade da versão inicial
- Não redesenhar a interface sem solicitação explícita
- O hero deve permanecer centralizado
- “Lele Matoos” no hero deve usar Imperial Script e rosa choque
- O visual deve continuar old school, feminino, vintage e clean

## Regras de implementação
- Componentes pequenos e reutilizáveis
- Dados mockados separados dos componentes
- Código fácil de alterar futuramente
- Evitar abstrações desnecessárias
- Evitar hardcode excessivo em page.tsx
- Preservar legibilidade

## Futuro
- O design pode mudar depois
- Estruture o código para facilitar mudanças de layout, estilo e conteúdo
